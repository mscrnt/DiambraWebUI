Installation Instructions

  • Linux: Follow the Docker Engine Install Guide.
  • Windows: Download and install Docker Desktop. Make sure WSL integration is enabled.
  • macOS: Download and install Docker Desktop.

Post-Installation Steps

  • Ensure your user is added to the Docker group (Linux only):
    sudo usermod -aG docker $USER
    Log out and log back in after running this command.
  • Verify the installation:
    docker --version
    docker run hello-world

If you encounter any issues, refer to the official Docker documentation.
